# mars-parse

## Install (Windows)
<ul>
  <li>Clone repo</li>
  <li><code>> cd mars-parse</code></li>
  <li><code>> python -m venv venv</code></li>
  <li><code>> venv\scripts\activate</code></li>
  <li><code>> pip install --user . </code></li>
</ul>

## Usage
<ul>
  <li><code>> cd src</code></li>
  <li><code>> python mars_parse.py --help</code></li>
  <li>Example: <code>> python mars_parse.py </code></li>
  <li><code>> pip install --user . </code></li>
</ul>
</ul>
